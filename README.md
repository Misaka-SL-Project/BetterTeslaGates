# Better Tesla Gates
Makes Tesla Gate inactive if holding an item in the list

## Config
```
# If the plugin is enabled or not.
  is_enabled: true
 # Available items that could disable the Tesla gate
  required_items:
  - KeycardMTFPrivate
  - KeycardContainmentEngineer
  - KeycardMTFOperative
  - KeycardMTFCaptain
  - KeycardContainmentEngineer
  - KeycardFacilityManager
  - KeycardO5
  ```
